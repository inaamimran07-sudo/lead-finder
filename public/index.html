⚙️ Setup Guide
Follow these steps once to activate lead searching.
1

Create a Google Cloud Account

Go to console.cloud.google.com and sign in with your Google account. It is free to sign up.

2

Create a Project

Click the project dropdown at the top, then New Project. Give it any name like "LeadForge" and click Create.

3

Enable the Places API

Go to APIs and Services, then Library. Search for Places API, click it and click Enable.

4

Set Up Billing (Free)

Go to Billing in the left menu and link a card. Google gives you $200 free credit every month — you will not be charged for normal usage. This step is required to use the API.

5

Get Your API Key

Go to APIs and Services, then Credentials, then Create Credentials, then API Key. Copy the key — it starts with AIza...

6

Paste Your API Key Here

Your key is stored securely on our server. Once saved, go to the Search tab.
